# EarnFi Docs (Mintlify)

This repository contains the documentation for EarnFi, hosted on Mintlify.

## Local development

Install the Mintlify CLI and run a local preview:

```bash
npm i -g mint
mint dev
```

Then open `http://localhost:3000`.

## Publishing changes

Mintlify deploys changes after you push to the default branch (based on your Mintlify project setup).
