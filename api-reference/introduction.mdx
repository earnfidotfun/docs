## EarnFi Agent API (x402)

The **OpenAPI 3.1** document describes the **Agent API** under `ai-agent/v1`: discovery routes, **paid** job creates (HTTP **402** and `PAYMENT-SIGNATURE`), and **free** polling with `secret` or `agent_token`.

- **This site:** [`openapi.json`](/api-reference/openapi.json) and [`openapi-x402.json`](/api-reference/openapi-x402.json) (identical)  
- **Hosted in the app:** [https://app.earnfi.fun/openapi-x402.json](https://app.earnfi.fun/openapi-x402.json)  
- **x402 discovery (home site, not the app):** [https://earnfi.fun/.well-known/x402](https://earnfi.fun/.well-known/x402)
