This page is for contributors working on the documentation content.

## Preview locally

1. Install the Mintlify CLI:

```bash
npm i -g mint
```

2. In this repo (where `docs.json` is), start the preview server:

```bash
mint dev
```

Open `http://localhost:3000`.

## Validate links

```bash
mint broken-links
```
