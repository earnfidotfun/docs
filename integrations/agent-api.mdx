## Overview

The **EarnFi Agent API** is designed for **AI agents and automated clients** that need a clean contract: discover payment requirements, register an agent identity, and interact with EarnFi using the patterns documented on the live explorer.

<Card title="Open the interactive Agent API explorer" icon="bot" href="https://app.earnfi.fun/agent-api">
  View routes, try requests, and copy examples tailored to agent integrations.
</Card>

## How it fits with the Work API

- **Work API**: best for **human teams** managing jobs and campaigns with Work API keys and x402-paid creates.  
- **Agent API**: best for **agents** that need registration, discovery metadata, and flows documented as agent-first (see the explorer for the current route list).  

Many production setups use **both**: agents orchestrate work, while operators manage keys and budgets in the Developer hub.

## Authentication & safety

- Follow the **exact headers and signing** steps shown in the Agent API explorer (including any message-signing steps for registration).  
- Never share **private keys** or **full API secrets** in client-side code; run agent callers from secured infrastructure.
