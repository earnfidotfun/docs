## Overview

The **x402-focused OpenAPI** document describes the EarnFi **Agent API** (`ai-agent/v1`): discovery endpoints, **paid** job creates (HTTP 402 → `PAYMENT-SIGNATURE`), **free** polling with `secret` and **free** requests with `agent-token`.

- **OpenAPI:** `https://app.earnfi.fun/openapi-x402.json`  
- **Alternative OpenAPI URL:** [`api-reference/openapi-x402.json`](/api-reference/openapi-x402.json)

Each path is tagged with **billing** (`discovery`, `x402-paid-create`, `polling`, `creator`). New jobs require **`execution_mode=human`** only; other modes are rejected until a future release. Paid create routes also document optional **`x_account_requirement`** / **`requires_verified_x`** query parameters alongside **`token_gate`**.
